# Background Images Folder

Place your background image here:
- `background.jpg` - for static backgrounds
- `background.gif` - for animated backgrounds
- `background.png` - also supported

The image will automatically be used across all pages.

## Quick Start
1. Add your image to this folder
2. Rename it to `background.jpg` or `background.gif`
3. Restart the dev server
4. Adjust opacity in `src/app/layout.tsx` if needed (line ~24)

For detailed instructions, see: `/BACKGROUND_SETUP.md`
