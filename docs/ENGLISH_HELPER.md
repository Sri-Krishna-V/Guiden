## Best Practices

### For Best Results
1. **Use headphones** - Prevent echo and feedback
2. **Good lighting** - For better video quality
3. **Quiet environment** - Reduce background noise
4. **Speak clearly** - Improve recognition accuracy
5. **Practice regularly** - Build consistency

### Common Issues

**Microphone not working?**
- Check browser permissions
- Ensure microphone is not muted
- Try refreshing the page

**Speech not recognized?**
- Speak more clearly and slowly
- Check microphone volume
- Ensure browser supports Web Speech API

**AI not responding?**
- Check internet connection (needed for TTS)
- Wait for AI speaking indicator to finish
- Submit your speech manually if needed
