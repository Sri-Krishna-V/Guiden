# **App Name**: CareerLens

## Core Features:

- AI Career Recommendation: Leverage user profile and preferences to generate top career recommendations with reasons, missing skills, learning plan, and resources using an AI tool.
- Skill Gap Analysis: Compare user skills with target role skills to identify overlaps, missing skills, and suggested learning order.
- Personalized Roadmap: Create a 3-month weekly learning plan with top free/paid resources based on the career recommendation.
- Resume Builder: Generate a clean, professional resume in JSON format based on user profile information.
- Interview Preparation: Generate interview questions (easy/medium/hard) with model answers for a specific career role, using an AI tool.
- User Profile Management: Allow users to create and manage their profiles, including education, experience, skills, and interests.
- Assessment Dashboard: Display assessment results and career recommendations in an intuitive dashboard.

## Style Guidelines:

- Primary color: Soft sky blue (#6FBFF2), evoking a sense of trust and aspiration. Hex code was converted from HSL values of hue 204, saturation 61, and lightness 68.
- Background color: Very light blue (#F2F9FF), providing a clean and calming backdrop. Hex code was converted from HSL values of hue 204, saturation 30, and lightness 97.
- Accent color: Muted violet (#A680FF), for highlighting key actions. Hex code was converted from HSL values of hue 257, saturation 100, and lightness 75.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and easily readable interface.
- Use modern, minimalist icons to represent different career paths and skills.
- Employ a clean and intuitive layout with clear sections for user profiles, assessments, and recommendations.
- Subtle animations for loading indicators and transitions to enhance user engagement.
